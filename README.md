# Album-Panini-Mundial-2026
Aplicación web full-stack que simula un álbum digital de figuritas (láminas) del Mundial 2026, inspirado en las clásicas colecciones Panini. Permite a los usuarios registrarse, iniciar sesión, explorar el catálogo completo de láminas y gestionar su propio inventario (agregar, pegar y eliminar láminas).
